#### 步骤
- 目标分析
- 数据清洗+可视化探索
- 加载停用词，并使用结巴分词
- 创建分类模型
- 模型训练

<br>
利用朴素贝叶斯进行新浪新闻的分类：<br>
其中每个文件夹下存放一批txt文件，代表该类型的所有样本新闻<br>
对应中，在ClassList.txt中存在对应文件夹名称所属新闻类型
<br>
因此：数据清洗步骤可省略，可视化步骤可使用词云显示每个类型新闻的词云情况。
<br>

**重点：中文分词的练习 + 词云可视化**
 
 #### 特征选择
 
 选择重要的词语特征作为每个类别的分类标准<br>
 这里需要提前去掉停用词（介词、标点等）
 
 **重点：如何去掉停用词**
 
 1. 读取所有数据（通过结巴进行分词），读取所有标签（通过标签分类获取文件夹名称对应的标签）
 2. 数据+标签进行对应
 3. 可视化，并加载停用词后可视化
 4. 选择模型
 5. 模型参数调整
